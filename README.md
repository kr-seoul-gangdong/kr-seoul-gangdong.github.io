# 강동구 · 서울특별시

> 강동구 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-seoul-gangdong.github.io/**

강동구는 한강이 서울을 빠져나가기 직전 마지막으로 감싸 도는 서울의 동쪽 관문으로, 기원전 4000년경 신석기인이 빗살무늬토기를 빚으며 첫 정착지를 이룬 암사동 유적의 고장입니다. 6천 년 전 강변 취락에서 시작된 삶의 흔적이 오늘날 1만 2천여 세대 규모의 초대형 아파트 단지 올림픽파크포레온과 한 지역 안에 공존하는, 선사와 첨단이 겹쳐진 독특한 도시입니다. 매화(구화)와 잣나무(구목), 종달새(구조)를 상징으로 삼아 강변의 온화함과 생태의 푸르름을 함께 담아냈습니다. 천호동 로데오거리의 활기, 고덕비즈밸리의 자족 업무단지, 강동경희대병원을 중심으로 한 의료 인프라가 어우러져 주거·상업·산업이 균형을 이룹니다. 서울 25개 자치구 가운데 가장 늦게까지 넓은 미개발지를 품고 있어 재건축과 도시재생으로 빠르게 변모하는 현재진행형의 구이기도 합니다.
